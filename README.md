## 프로젝트 실행 방법

### 1. git repository clone
```
git clone https://github.com/JunYoung7/triple-assignment.git
```

### 2. npm install
```
npm install
```

### 3. 프로젝트 실행
```
npm start
```

### 4. storybook 테스트
```
npm run storybook
```
